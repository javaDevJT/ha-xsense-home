# X-Sense Home HACS Documentation

This directory stores persistent project knowledge for the X-Sense Home Home Assistant/HACS integration effort.

## Structure

- `publishing/` - release notes, HACS publishing notes, and validation records for tags/releases.
- `research/` - dated investigation notes, source inventories, feasibility findings, and implementation recommendations.

## How To Use These Notes

- Start with the newest file in `research/` when resuming integration work.
- Check `publishing/` before tagging or releasing so release notes and validation status stay aligned with GitHub.
- Treat linked upstream repositories and product pages as volatile; re-check versions before implementing or releasing.
- For Home Assistant/HACS validation, verify both repository state and the visible Home Assistant/HACS runtime state after install or update.
